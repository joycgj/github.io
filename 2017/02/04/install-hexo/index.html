Hexo是一个简单地、轻量地、基于Node的一个静态博客框架，可以方便的生成静态网页托管在github和Heroku上。引用Hexo作者 @tommy351 的话：

快速、简单且功能强大的Node.js博客框架。A fast, simple & powerful blog framework, powered by Node.js.

要使用Hexo，需要在你的系统中支持Nodejs和Git。
